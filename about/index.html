    關於我

    我是莊昊耘（Milan），NLP 工程師與 LLM 開發者，擁有國立政治大學語言學研究所計算語言學碩士背景，並曾交換至德國杜賓根大學深造。畢業於國立成功大學外國語文學系與工業暨資訊管理學系雙學士。

    具備完整 NLP 技術實務經驗，涵蓋 BERT 微調、序列標記、AI Agent 開發，以及 LLM Prompt Engineering 與 RAG 系統設計。曾獲國科會補助赴荷蘭葛羅寧根大學發表國際學術論文。

    不甘侷限於單一領域，從外文系出發，自學 Python、爬蟲技術與機器學習，積極參與跨領域黑客松競賽——包含 SAS 人工智慧競賽與法遵法律科技黑客松，帶領十人跨領域團隊，兩者皆進入全國決賽。

    我熱衷於以程式語言探索文本資料背後的邏輯與隱藏架構，是一名站在人文出發點的資訊人，對任何新技術都抱持「學了就會」的態度。
